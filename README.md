# Dog-Walking-App
### Roles: Dog Walker, Dog Owner
### Use case:
#####  Dog Owner
    1. Create task
    2. Cancel task
    3. Confirm payment
    4. Rating
#####  Dog Walker
    1. Browse tasks.
    2. Accept tasks.
    3. Start a task & create a record
    4. Finish a task & submit a record
    5. receive payment
    6.Rating
